========================================
SUPPORTED RELEASES
----------------------------------------

Currently we are targetting Perl 5.08 or newer. This brings back the supported 
distro to around 10 years old or newer, roughly speaking. Note that inxi bash
has a similar requirement, in that it does only supports:

bash >= 3.0 (July 27, 2004). As you can see from the Distro release dates, you
can expect to see Perl 5.10 in All distros as of 1010, which is a touch too 
recent. 

Note however that Perl 5.10 introduces switch, say, pragma. But we don't really 
need that, and there's no issues using 5.08 as long as we keep that in mind.

https://perldoc.perl.org/feature.html

https://en.wikipedia.org/wiki/Perl

PACKAGES
----------------------------------------

NOTE: perl 5.008 needs package inside brackets.
I believe 5.010 introduced option to have it outside brackets as you'd expect

that is:
5.08:
{
package MyPackage;
...
}
5.10:
package MyPackage;
{
....
}

----------------------------------------
qr/../i
----------------------------------------

Precompile regex only adds i at 5.014, not sure about other values, but just 
avoid using other switches there to be safe.

----------------------------------------
//=
----------------------------------------

Short test for if not defined assign value. Similar to ||= except it doesn't
work on Perl 5.008. I do not know when it was added, but don't use it without
testing an original install of that Perl version, if we ever go to  5.10 or 
newer to get the added features given/say.

----------------------------------------
GIVEN/WHEN
----------------------------------------

<<<
You can switch to the built-in given/when if using Perl 5.10+. If stuck with 
Perl 5.8, I'd just use an if/elsif block instead. 
>>>
http://shop.oreilly.com/product/9780596001735.do

Value Switches - Use table look-up in preference to cascaded equality tests 

use 5.010;

given($value) {
    when(3) { say "Three"; }
    when(7) { say "Seven"; }
    when(9) { say "Nine"; }
    default { say "None of the expected values"; }
}
https://perlmaven.com/switching-in-perl-5.10

use 5.010;

given($value) {
    when(10) {
        say "Number 10";
    }
    when([11, 23, 48]) {
        say "In the list";
    }
    when(/^\d+$/) {
        say "digits only";
    }
    when(\&is_number) {
        say "Is number";
    }
    default {
        say "None of the above";
    }
}

sub is_number {
    return $_[0] =~ /\d/ and $_[0] =~ /^[+-]?\d*\.\d*$/;
}


http://perl.find-info.ru/perl/028/perlbp-chp-6-sect-16.html
